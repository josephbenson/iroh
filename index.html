The Problem

Talent

  • Nigerian engineering talent is high-agency but invisible to global capital
  • No single company has the hardware + software + AI depth for niche problems
  • Existing programs get bypassed — talent gets poached without platform value accruing

Data

  • No reliable ground-truth data on the Nigerian market — input costs, agent numbers, infrastructure
  • Hidden resources exist (government labs, mini-grids, long-term mortgages) but are unmapped
  • Without this data, diaspora capital and foreign investors can't make informed decisions

How It Works

  1. Layer 1

    Data Collection

    Organized youth groups (NYSC cohorts, university clubs, JCI) collect narrow, specific datasets. Incentive: diaspora network access, not cash.

  2. Layer 2

    Skills Pipeline

    Data collectors feed into the fellowship. Engineers qualify via a curated free learning path — no curriculum to build, just a filter.

  3. Layer 3

    R&D Marketplace

    Companies post real problems with attached budgets. Engineers deliver tangible outputs. Universities supervise for QA.
